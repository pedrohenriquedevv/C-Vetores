# Lista de Exercícios: Vetores, Strings e Algoritmos em C 🚀

Este repositório contém as minhas resoluções para uma lista de exercícios acadêmicos focada em lógica de programação, manipulação de estruturas de dados homogêneas (**Vetores** e **Strings**) e algoritmos de ordenação e busca na linguagem **C**.

O objetivo deste projeto é consolidar e aprimorar minha base técnica e teórica em algoritmos, gerenciamento manual de arrays e manipulação de cadeias de caracteres sem o uso de abstrações de alto nível.

## 🛠️ Tecnologias e Ferramentas
* **Ambiente de Desenvolvimento:** VS Code

## 📌 Índice de Exercícios

O repositório está organizado de forma que cada arquivo corresponda exatamente aos exercícios da lista:

### Manipulação de Vetores (Numéricos)
* **`exe1.c`**: Lê um vetor de 12 posições e conta quantos números são pares e quantos são ímpares.
* **`exe2.c`**: Lê 10 números e copia apenas os valores positivos para outro vetor.
* **`exe3.c`**: Lê um vetor de 20 elements inteiros e mostra os elementos na ordem inversa.
* **`exe4.c`**: Lê dois vetores A e B de 10 posições e gera um terceiro vetor contendo a soma dos elementos correspondentes.
* **`exe5.c`**: Lê um vetor com 15 números e substitui os números negativos por 0.
* **`exe6.c`**: Lê 10 valores e verifica se um número informado pelo usuário existe no vetor (se existir, indica a posição).
* **`exe7.c`**: Lê um vetor e informa quantas vezes um determinado número aparece.
* **`exe8.c`**: Lê 10 números inteiros e ordena o vetor em ordem crescent.
* **`exe9.c`**: Lê um vetor de 20 posições e remove os elementos repetidos.
* **`exe10.c`**: Lê 12 números e divide os elementos em dois vetores (um contendo números pares e outro ímpares).
* **`exe11.c`**: Lê dois vetores de 10 posições e verifica se eles são iguais.
* **`exe13.c`**: Lê um vetor e rotaciona os elementos uma posição para a direita.

### Manipulação de Strings (Char Arrays)
* **`exe14.c`**: Lê uma string e mostra a frase invertida.
* **`exe15.c`**: Lê duas palavras e verifica se elas são iguais.
* **`exe16.c`**: Lê uma frase e substitui todas as letras “a” por “@”.
* **`exe17.c`**: Lê uma palavra e verifica se ela é um palíndromo (tratando espaços em branco).
* **`exe18.c`**: Lê uma frase e conta a quantidade de vogais, consoantes, números e caracteres especiais.
* **`exe19.c`**: Lê uma string e remove todos os espaços em branco.
* **`exe20.c`**: Lê um nome completo e mostra apenas o primeiro nome.
